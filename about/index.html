個人介紹

不善交際，不常出門，非常孤僻 (?) 的工程師
目前主要寫 Web 後端

追求易於閱讀的程式碼，喜歡去研究一些程式的寫法，讓程式可以一眼看出意圖並且不會有過多冗長且不重要的資訊

閒暇時除了玩些遊戲，偶爾會抽一點時間來寫寫楓之谷伺服器當興趣

能力

對於常用的語言如 Typescript、Java、C#、PHP、Python、C/C++ 等都有基礎的認識，能夠做一些小程式開發，語言部份不是問題。

Web 後端部份主要使用 Node.js、SpringBoot。過去也用過 PHP 的 CodeIgniter 跟 Laravel，但已經有幾年沒碰了。

本身有接觸一些 Gitlab CI/CD、Docker、K8S 等 DevOps 領域的東西，也能夠寫一些簡單的 Shell 腳本。

目前的技能樹比較雜，通常都是工作上有需要時再去學習

[雜談] 2025-07-13 - 在 MSW 的第 3 個月

發表於 2025-07-13 分類於 雜談 ， MapleStory Worlds

最近依然在 MSW 上隨便玩玩
但開始缺少動力了

中文區域才開放不到 3 個月，就有各種事情發生
不論是最初的外掛進駐、伺服器連線問題，再到現在創作者的心血被剽竊
玩家對事件的反應，以及官方消極的態度，都讓我對這個平台感到非常失望

開發者社群則是一片死寂…

因為最近都在 MSW 的關係，所以也只有 MSW 的東西能講 :3
大概分享一下最近發生的事情，跟一些心得

自製 PlayerAnimationComponent

我最近一直都在做角色動畫
沒為什麼，只是我真的不知道我要做啥 🙃

因為 MSW 只提供基礎的角色動畫，但從正服可以發現，現在技能施展時都會有一些角色動作
所以我就寫了一個 PlayerAnimationComponent，可以執行自訂動作，並且處理玩家間的動作同步

後來又寫了編輯器，嘗試解決角色動作設計的難題
因為官方素材在 MSW 裡面，如果要設計動作，一直重測實在是太麻煩了
作為嘗試，我使用 Editor 相關事件 寫了一個角色動作編輯器，方便設計自訂動作

相關元件以及編輯器目前已經上架成一個 Demo 世界開源
歡迎有興趣的人透過重製 (Remake) 功能把世界複製回去使用：自訂角色動作: 楓之谷世界

後記

MSW 在編輯模式下，可以用的事件非常的少
拖放、滾輪、鍵盤全都沒辦法實現，在這個情況下寫編輯器的投資報酬率有夠低…

發布當天，剛好紙娃娃盜取風波正在上演，所以還被幾個人諮詢問能不能摳圖 = =
如果開了留言，可能會出現一堆有的沒的留言，我沒辦法負荷…

MSW 紙娃娃盜圖事件

這似乎是一直存在 MSW 的問題
大概在 2~3 周前，開始出現有造型被複製的問題
還有人高調說要公佈提取器，雖然最後沒有發布，估計是騙人的

楓之谷的紙娃娃其實就是單純的 2D 圖片
起先，我猜想可能是 Cache 被濫用
因為 MSW 的快取並沒有做任何處理，可以說是把圖片、聲音等素材檔案赤裸裸的公開給所有使用者

後來我也去看過 Cache 的內容，確實沒有任何處理，任何人，甚至連程式小白都可以輕易的取出圖片檔案

整體檔案結構並不難，花了大概 2 天的時間，整份檔案就解得差不多了
不過目前還沒看到有人拆檔，還是以取圖片為大宗，要是哪一天檔案拆包也被公開，這個平台大概就完了…

目前為止對 MSW 的想法

上次的雜談也有一樣的段落
跟上次比，我對 MSW 的信心與熱情都降到一個低點

紙娃娃創作社群被盜圖事件搞得烏煙瘴氣
加上官方遲遲不公佈提領收益確切的時間日期
使得中文區的創作人數非常的少

這個平台目前對創作者極度不友善
打從一開始， Nexon 30%，Toben 35% ，共 65% 的收益抽成就已經讓很多人不爽了
現在金額達到還沒辦法提領，中文區已經沒看到討論遊戲創作的人了
連創作紙娃娃都會被盜取，創作者可以說一點保障都沒有

部份玩家的素質也極度堪憂
各種不看頻道名稱就亂問、到處叫賣、現金交易的
不然就是在 MSW 官方社群上瘋狂漫罵 Artale 官方毫無作為，搞得好像這個平台只有 Artale 一樣
創作者頻道也淪為叫賣場，雖然官方後面開始刪對話，給頻道一個清靜，然後裡面就幾乎沒對話了…

盜圖風波的部份，意外的有很多人支持
甚至特別拿二創出來把盜取這個行為合理化

這樣的玩家素質，我自己就悲觀的覺得這個平台不會再有更多遊戲了，至少中文圈是這樣

暫時不會在 MSW 上寫 Code 了

MSW 的事情實在太多了
中文區才開放不到 3 個月，整個平台就快被搞壞了
每天看著群組裡面的負面訊息，以及無理取鬧的玩家要求，真的太累了…

中文創作者社群一片死寂，我只能往英文區、韓文區看感興趣的內容跟問題
感覺大家都各作各的，也沒看到有誰開發什麼好用的工具鍊 (韓文區也許有，我需要跳 VPN 去看看)
編輯器自全球開放之後就還沒有什麼更新

感覺比起以前的創作、幻想，現在大家更在乎效率跟錢
人們進入 Artale 除了圓小時候的夢，剩下的就只是無止盡的練等跟賺錢
沒人在乎這個遊戲創造的氛圍，也沒人在乎這個遊戲的實質內容，就只是無止盡的練等、打王，然後買賣，把舊楓谷活得像新楓之谷一樣

我喜歡楓之谷，是因為他的可塑性很高
雖然紙娃娃的動作有限，但還是可以激發一些想像
以前總是會幻想一些自製場景，自製技能
但現在…好像只剩數字而已…

在休息之前，我還會更新一下 PlayerAnimationComponent 的編輯器
希望能減輕新增動作時痛苦的感覺
之後我就不會再做任何更新了也不會再 MSW 上寫 Code
還是把它當成一個單純的遊戲平台就好，靠官方的原創世界豐富內容了👍
